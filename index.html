This Project

This project was built to integrate the Bootstrap framework and jQuery with the Electron framework.

Built with the following parameters:

  • Built ontop of the electron-quick-start application
  • Fixes Electron require() is not defined in client rederer
  • Fixes Electron jQuery/$ is not defined in client renderer
  • Contains examples of communicating between ipcMain and ipcRenderer
To Use

To clone and run this repository you'll need Git and Node.js (which comes with npm) installed on your computer. From your command line:

Clone this repositorygit clone https://github.com/electron/electron-quick-start
Go into the repositorycd electron-quick-start
Install dependencies npm install
Run the applicationnpm start
